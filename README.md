## A vscode extension for book reading
support .txt file only


## usage
+ install extension
+ run command startLoaf, at the first time, it's required to select a folder where you storage your books
+ then it will open a new file for reading `vscode-loaf.cpp`, if you are a cpp developer, this may not quit suit for you
+ use tab(nextPage) left(prevPage) to change page
+ use command `switch book` `search page` to change reading


## notice
+ use yarn to download node_modules, pnpm is not compatible with vsce
+ move all devDependencies to dependencies, make sure the vsix include modules.
